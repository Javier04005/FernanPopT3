FernanPop - Plataforma de Compra y Venta

FernanPop es una aplicación de consola desarrollada en Java que simula un mercado de artículos de segunda mano. Permite a los usuarios gestionar su perfil, poner productos a la venta, consultar el catálogo disponible y revisar sus historiales de transacciones.
